# Link Vercel to this repo (one click)

**Use this link** — it opens Vercel with [buckeye7066/smalltownbigstage](https://github.com/buckeye7066/smalltownbigstage) pre-selected:

## Import link

**[Import smalltownbigstage into Vercel](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fbuckeye7066%2Fsmalltownbigstage)**

1. Click the link above (log in to Vercel if asked).
2. On the import screen:
   - **Root Directory:** click **Edit** → type **`frontend`** → Save.
   - **Environment Variable:** add **`NEXT_PUBLIC_API_URL`** = your Railway backend URL (e.g. `https://....up.railway.app`). If you don’t have it yet, add it later in Settings → Environment Variables.
3. Click **Deploy**.
4. After deploy: **Settings** → **Domains** → add **www.smalltownbigstage.org** and **smalltownbigstage.org**.

After that, this repo is linked to Vercel and the domain will show the site from your file.
